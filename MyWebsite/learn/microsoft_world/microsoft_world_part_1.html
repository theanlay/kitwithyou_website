ពេលនេះយើងនិងនិយាយពីមុខរបស់font ទាំង៦សិន។ ដែលក្នុងនេះមានដូចជា:

  1. B:Bold វាធ្វើអោយអក្សរដិត shortcut key is (Ctrl+b)
  2. I:Italic វាធ្វើអោយអក្សទ្រេត shortcut key is (Ctrl+i)
  3. U:Underline វាធ្វើអោយអក្សរមានបន្ទាត់គូរពីក្រោម shortcut key is (Ctrl+U)
  4. abc:strikethrough វាបានធ្វើអក្សរយើងមានស្មាគូរកណ្ដាល
  5. X2:Subscript បង្កើតនិទ្ទសន៏ shortcut key(Crtl+=)
  6. X2:Superscript បង្កើតស្វ័យគុណ shortcut key is (Ctrl+Shift++)
